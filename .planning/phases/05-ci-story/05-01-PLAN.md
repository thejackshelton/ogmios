# Phase 5 Plan 01 — Tart image pipeline

## Objective

Ship the Packer + Ansible pipeline that produces the reference pre-baked
macOS images (`ghcr.io/shoki/macos-vo-ready:<macos>`) with VO AppleScript
enabled, Accessibility + Automation TCC grants pre-applied, Node 24 + pnpm
10 + Playwright installed, and flake-causing services disabled.

## Tasks

1. `infra/tart/packer/sonoma.pkr.hcl` — Packer template with
   cirruslabs/packer-plugin-tart, base image
   `ghcr.io/cirruslabs/macos-sonoma-xcode:latest`, ansible provisioner,
   slim-vs-@full variant handling.
2. `infra/tart/packer/sequoia.pkr.hcl` — same shape, macOS 15 base.
3. `infra/tart/packer/tahoe.pkr.hcl` — same shape, macOS 26 base with
   header note explaining the base may not yet exist.
4. `infra/tart/ansible/playbook.yml` — 6 stanzas:
   VO AppleScript, TCC grant insertion, background-app quit, service
   disable, toolchain install, image marker.
5. `infra/tart/ansible/vars.yml` — bundle IDs, Node/pnpm versions,
   background-app list, services-to-disable list.
6. `infra/tart/scripts/tcc-grant.sh` — tccutil-style TCC.db upsert, with
   VM-guard and SIP-off check.
7. `infra/tart/README.md` — architecture explanation, local-build
   instructions, slim vs @full, known sharp edges.

## Verification

- Files exist at paths above
- `infra/tart/scripts/tcc-grant.sh` is chmod +x
- Packer HCL is syntactically valid (requires `packer init` to validate
  fully; deferred to tart publish workflow run)

## Requirements coverage

- CI-01: reference tart image published (image produced by this pipeline;
  publish workflow is Plan 03)
- CI-02: reproducibly built with Packer + Ansible
