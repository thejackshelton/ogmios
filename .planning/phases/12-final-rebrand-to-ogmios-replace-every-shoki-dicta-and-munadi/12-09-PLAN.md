```markdown
## 0.1.1 — 2026-04-18

### Renamed

- The library is now called **Ogmios** (from `munadi`, `dicta`, `@shoki/core`, `shoki`).
  Final name. After three attempts landed on npm's similarity filter (vs `shiki`,
  `shika`, `minami`), Ogmios (the Gaulish god of eloquence) is mythologically
  grounded, distinctive, and unambiguous.
- `npm install ogmios` (unscoped) + `@ogmios/binding-darwin-arm64` (platform binding).
- CLI is `ogmios doctor`, `npx ogmios setup`.
- Helper apps rename `MunadiRunner.app` / `MunadiSetup.app` -> `OgmiosRunner.app` /
  `OgmiosSetup.app` with bundle IDs `org.ogmios.runner` / `org.ogmios.setup`.
- State dir `~/.munadi/` -> `~/.ogmios/`. `ogmios doctor` detects any of
  `~/.shoki/` / `~/.dicta/` / `~/.munadi/` on first run and prints a one-line
  notice with a safe `rm -rf` command.
- Breaking: all `MunadiX` / `ShokiX` error classes are now `OgmiosX`. Factory
  `munadiVitest()` -> `ogmiosVitest()`. Plugin detection needle `munadi/vitest/browser`
  -> `ogmios/vitest/browser`. Env vars `MUNADI_*` / `SHOKI_*` -> `OGMIOS_*`.

### Deprecated on npm

- `dicta@0.1.0` — renamed to `ogmios`. Install: `npm install ogmios`.
- `@shoki/binding-darwin-arm64@*` — renamed to `@ogmios/binding-darwin-arm64`.

*(If `munadi` was never successfully published, there is no deprecation there.)*
```
