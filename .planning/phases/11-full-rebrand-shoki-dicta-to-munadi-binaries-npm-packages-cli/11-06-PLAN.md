## [0.1.1] ŌĆö 2026-04-XX

### Renamed to Munadi

Fourth and final name pivot. The package is now `munadi` (Arabic: ┘ģ┘Å┘åž¦ž»┘Ŗ, "herald" / "one who announces"). Rebrand history:
1. Originally `shoki` ŌĆö blocked by npm's E403 anti-typosquatting filter vs `shiki`
2. `@shoki/core` ŌĆö `@shoki` org creation was denied
3. `dicta` ŌĆö Latin "things said", unscoped, published as v0.1.0
4. `munadi` ŌĆö Arabic for "herald". Thematically perfect for a screen-reader announcement library. This rebrand is COMPLETE ŌĆö helper app, binding packages, env vars, state dir, plist keys, Zig source, infra, repo URL all renamed.

### Breaking

- npm package: install `munadi`. `dicta` is deprecated (pointer to `munadi`).
- Binding packages: `@shoki/binding-darwin-*` deprecated. Install resolves `@munadi/binding-darwin-*` via `munadi`'s optionalDependencies.
- CLI bin: `munadi doctor`, `npx munadi setup`, `munadi info`, `munadi restore-vo-settings`. `dicta` CLI no longer ships.
- Error classes: every `Shoki*` / `Dicta*` class is now `Munadi*`. Update catch-class names.
- Env vars: `SHOKI_HELPER_PATH` ŌåÆ `MUNADI_HELPER_PATH`, `SHOKI_AX_TARGET_PID` ŌåÆ `MUNADI_AX_TARGET_PID`, `SHOKI_INTEGRATION` ŌåÆ `MUNADI_INTEGRATION`, `SHOKI_SNAPSHOT_PATH` ŌåÆ `MUNADI_SNAPSHOT_PATH`, `SHOKI_NATIVE_BUILT` ŌåÆ `MUNADI_NATIVE_BUILT`.
- State dir: `~/.shoki/` and `~/.dicta/` ŌåÆ `~/.munadi/`. Legacy dirs: `munadi doctor` on first run detects them and prints `rm -rf ~/.shoki ~/.dicta` (clean break; no auto-migrate for pre-alpha).
