Rename the tart/Packer/Ansible infrastructure from Shoki to Munadi: image labels, VM names, Ansible variable names, TCC-grant script filename + defaults, VM marker file, and README prose.

Output: `infra/tart/**` fully rebranded. Packer images would build with Munadi labels (no CI run required in this plan — the Ansible/Packer scripts are the contract, image rebuild is a CI/release concern).
</objective>
