# ogmios-darwin-x64

Native addon for Ogmios on macOS Intel.

This package is installed automatically by `ogmios` on matching platforms. You should not depend on it directly.

**Contents:** N-API addon compiled from Zig 0.16.0 via napi-zig. A signed `OgmiosRunner.app` helper bundle is downloaded on first run and used as the TCC trust anchor (see PROJECT.md Key Decisions).
